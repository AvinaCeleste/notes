Notes

Colors can be specified using RGB values, HEX values, HSL values, RGBA values and HSLA values. RGB VALUES: rgb(red, green, blue)
HEX VALUES: #rrggbb
HSL VALUES: hsl(hue, saturation, lightness)
RGBA VALUES: rgb(red, green, blue, alpha(opacity))
HSLA VALUES: hsl(hue, saturation, lightness, alpha(opacity))
