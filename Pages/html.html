Notes

HTML stands for Hyper Text Markup Language. Element names are surrounded by angle brakets to create tags. <tagname>content goes here...</tagname> HTML tags usually come with a start tag and an end tag. The end tag is like the start tag but with a forwards slash before the element name.
HTML documents must start with a document declaration. <!DOCTYPE html> The html tag is a container for your HTML code. The visible part of your code is contained in a body tag. <h1>HEADER</h1>
<p>PARAGRAPH</p>
<a href="#">LINKS</a>
<img src="#" alt="Alternative Text">IMAGES</img>
<button>BUTTON</button>
<ul>UN-ORDERED LIST</ul>
<ol>ORDERED LIST
<li>LIST ITEM 1</li>
<li>LIST ITEM 2</li>
<li>LIST ITEM 3</li>
</ol>
<pre>PREFORMATTED TEXT</pre> HTML elements can be nested within others, as seen in the ordered list example above. Elements that do not contain content are called empty elements. <b>

Attributes

Attributes provide additional information about an element. Attributes are always defined in the start tag. Using the style attribute, CSS elements can be used in an html document. <body style="color:blue;"></body>

Metadata

Metadata is undisplayed data about the html document. It typically defines a document title, character set, stylesheets, links and other metadata. <meta charset="UTF-8">

Comments in code can be written using: <!--THIS IS A COMMENT-->
